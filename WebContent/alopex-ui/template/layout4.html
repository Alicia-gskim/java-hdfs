일반 컨텐츠 UI Layout Grid 적용법

일반 컨텐츠도 Layout 적용시와 동일하게 적용합니다.
  • UI Layout Grid 사용전 Markup이 block 요소이면 'Grid-sm-12'로 전체를 감싸지 않아도 됩니다. 기본적으로 float: left 가 적용되어 있습니다.
    (여기서는 없다고 가정하고 예제를 적용하기로 합니다.)
  • Nesting 될 요소가 복잡하다면 'Grid-sm-12'로 Nesting 부모요소를 정의 해줍니다.
